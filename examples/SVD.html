SVD Example: Image Compression¶

We can express any matrix $A \in \mathbb{R}^{m \times n}$ in a decomposed form as:

$A = U \Sigma V$

The matrices $U\in \mathbb{R}^{m \times n}$ and $V\in \mathbb{R}^{n \times n}$ are unitary. This means $U^T U = U U ^T = I$, and similarly for $V$.

The matrix $\Sigma \in \mathbb{R}^{n \times n}$ is a diagonal matrix whose elements are $\sigma_1 > \sigma_2 > ... > \sigma_n$. The convention of SVD is to list the singular values in descending order.

The rank of the matrix $A$ is the number of singular values greater than zero.

We can calcualte two different matrix norms from singular values as $||A||_{2}|| = \sigma_{1}$ and $||A||_{F} = \sqrt{\sigma_{1}^{2} + \sigma_{2}^{2} + ... + \sigma_{n}^{2}}$.

Relationship to eigenvalues ¶

For symmetric matrices $A$, we have $A A^T = U \Sigma V V^T \Sigma U^T$ $A^T A = V^T \Sigma U^T U \Sigma V $

Using the properties of unitary matrices, this gives us: $A A^T = U \Sigma^2 U^T$ $A^T A = V^T \Sigma^2 V $

The eigenvalues of $AA^T$ and $A^T A$ are the singular values of $A$ squared. $\lambda{i} = \sigma{i}^2

Every matrix has an SVD. The singular values are uniquely determined. The vectors of $U$ and $V$ are determined up to a minus sign.

Relationship to Image Compression¶

The SVD allows us to use low-rank approximations of $A$. That is to say, we try to represent $A$ with an approximation by leveraging our knowledge of singular values.

By its construction, the SVD can be written as a sum of rank-one matrices.

$A = U \Sigma V = \sum_{j=1}^{n} \sigma_j u_j v_j$

We note now, that the rank of $A$ is determined by the non-zero singular values, so the above can be written as

$A = U \Sigma V = \sum_{j=1}^{r} \sigma_j u_j v_j$, where $r$ is the rank of $A$, that is to say, $\sigma_{r+1}, \sigma_{r+2}, ... , \sigma_{n} = 0 $.

If we perform the above summation, we recover the exact matrix $A$. But we can have a low-rank approximation if we only sum up to $k$ terms. Let us call the reduced matrix $A_{k}$.

$A_k = U \Sigma V = \sum_{j=1}^{k} \sigma_j u_j v_j$. That is the best rank-k approximation of $A$ we can obtain.

We note the following $||A - A_{k}|| = \sigma_{k+1}$ (using the property of matrix norms above).

In the Frobenius norm, we have $||A-A_{k}||_F = \sqrt{\sigma_{k+1}^{2} + \sigma_{k+2}^{2} + ... + \sigma_{r}^2}$.

We can see that using only 50 terms of the singular values, we have gotten back an image that looks perceivably close to the original image. But how much did we save?

The original matrix $A$ has $m\times n$ numbers that we must keep. How many terms do we need to hold to reduce using SVD?

$U$ will hold $m \times k$ terms, $\Sigma$ will hold $k $ terms, and $V$ will hold $k\times n$ terms.

This means a total of $ mk + k + kn = k(m+n+1)$ terms. As long as $k(m+n+1) < mn$, we have saved in terms of number of elements to keep track of.

Can we estimate mathematically how many singular values to keep to achieve a certain tolerance? Instead of just guessing how many singular values we need? Yes. We can estimate how many singular values we need to recover some fraction of the original matrix.

We can take the norm of the approximation divided by the norm of the original matrix to obtain the fraction $\eta_F = \frac{||A_{k}||_{F}}{||A||_{F}}$. We can also use the 2-norm as $\eta_2 = \frac{||A_{k}||_{2}}{||A||_{2}}$

$||A||_{f} = \sqrt{ \sum{\sigma_{i}^{2}}}$

Therefore, our fraction above can be written as $\eta = \frac{\sigma_{0}^{2} + \sigma_{1}^{2} + ... + \sigma_{k}^{2}}{\sigma_{0}^{2} + \sigma_{1}^{2} + ... + \sigma_{n}^{2}}$

Now if we set $\eta$ to a value of our choosing, we can figure out exactly how many singular values we need to keep.
